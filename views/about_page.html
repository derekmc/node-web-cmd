WebApp Terminal

About

Write simple web applications with a terminal user interface.

Creating web applications takes a lot of work. Typically, the user's work flow will be divided into several pages, and each of those pages must be designed and styled in a responsive way, to fit many different screens. Finally, the web site needs to be tested in a variety of browsers.

What if there were a simple way to write interactive web applications which avoided all that hassle? That's the primary goal of WebApp Terminal.

Terminal applications are the simplest programs to write and are learned by brand new programmers. They have a very simple interpretative design, where the program prompts for text input, and then prints text output. There is no worrying about graphics or layout or multiple pages. All the magic happens with standard in and standard out.

While terminal applications may not be the best choice for a mature or polished program, they are a great way to prototype and test an idea.

WebApp Terminal is a framework that hopes to combine the simplicity of terminal applications with the power of the web:

  • Global user interaction.
  • No installation or updates.
  • Secure and reliable remote handling of data and computation.
  • Linking to and from the rest of the web.

So go ahead, write terminal based games, dating apps, document editors, accounting tools, or cryptocurrencies. Nothing is stopping you.
